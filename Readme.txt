# dfDataTables

Ein leichtes, flexibles Tabellen-Plugin für WordPress.

## Funktionen
- Mehrere Tabellen verwalten
- CSV, SQL oder API als Datenquelle
- Doppelklick-Aktionen
- Export nach Excel, CSV, PDF
- Filterung und Suche je Spalte

## Installation
1. Hochladen nach `/wp-content/plugins/`
2. Aktivieren im WordPress-Backend
3. Tabellen erstellen und Shortcode `[dfdatatable id="XYZ"]` einfügen
